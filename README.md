This project explores how academic and career-related factors are associated with early-career outcomes. We analyzed categorical data to examine relationships between variables such as field of study, number of internships completed, job offers received and career satisfaction.

Research Questions
- Is there an association between the number of internships and job offers?
- Is career satisfaction different between STEM and non-STEM fields?
- Which log-linear model best explains the relationship among field of study, internships, and job offers?

Methods
- Categorical data analysis
- Association tests
- Log-linear modeling

Key Finding
The analysis showed a strong association between the number of internships completed and the likelihood of receiving job offers, suggesting that internship experience plays an important role in early-career outcomes.
